🛒 Suncart - Modern E-Commerce Platform
A sleek, high-performance e-commerce web application built with Next.js 15. Suncart provides a seamless shopping experience with real-time cart management, secure authentication, and a responsive UI.

🚀 Purpose
The primary goal of Suncart is to provide users with a fast and intuitive platform to browse products, manage their shopping carts, and handle their personal profiles. It focuses on modern web standards, optimized performance, and a clean user interface.

✨ Key Features
Next.js 15 App Router: Utilizing the latest React features for optimized routing and rendering.

Full-Stack Auth: Secure user authentication (Login, Register, Profile Management) powered by Better Auth.

Real-Time Cart Logic: * Add products once (preventing duplicates).

Increase/Decrease quantity within the cart.

Automatic subtotal and total price calculation.

Dynamic Product Routing: Specialized ViewDetails pages for every product.

Profile Management: Custom user profiles with the ability to edit names and profile pictures.

Responsive Design: Fully mobile-friendly UI built with Tailwind CSS.

Interactive Notifications: Real-time feedback using React-Toastify.

🛠️ Tech Stack & Packages Used
Core:
Next.js 15 (Framework)

React 19 (Library)

Tailwind CSS (Styling)

Authentication:
Better Auth (Comprehensive auth solution)

Icons & UI:
Lucide React (Modern, lightweight icons)

DaisyUI / HeroUI (Component libraries for sleek buttons and cards)

React-Toastify (For toast notifications)

State Management:
React Context API (For global Cart and Auth state)

📦 Installation & Setup
Clone the repository:

Bash
git clone https://github.com/your-username/suncart-next-app.git
Install dependencies:

Bash
npm install
Setup Environment Variables:
Create a .env.local file and add your database and auth credentials.

Run the development server:

Bash
npm run dev
Open the app:
Navigate to http://localhost:3000 in your browser.
